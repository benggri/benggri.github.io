Plugins

Extensions(Ctrl + Shift + X) 메뉴에서 Dev Containers 검색하시고 설치하세요

개발 준비

  • 먼저, Dev Containers 플러그인이 설치된 VS Code 를 열고 작업할 폴더를 만들고 여세요.
  • 저의 경우 workspace/vscode/프로젝트명 입니다
  • 그리고 Ctrl + Shift + P 를 누릅니다.
  • Dev Containers 를 검색하시고 Dev Containers: Add Dev Container Configuration Files... 메뉴를 선택합니다.
  • Java & PostgreSQL 를 검색하시고 선택합니다.
  • Java 버전은 최신 버전(2023년12월19일 기준) 21-bullseye 으로 선택하겠습니다.(참고로 Spring Boot 3.2.0은 java 17부터 됩니다.)
  • 그리고 아무것도 선택하지 말고 OK를 계속 누르세요.
  • 그러면 아래와 같은 폴더와 파일이 생성됩니다.
.devcontainer
  |_ devcontainer.json
  |_ docker-compose.yml
  |_ Dockerfile
